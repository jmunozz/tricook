# Tricook

Prototype of a shared meal planner and global shopping list.
Built with [Expo](https://expo.dev) so it can run on mobile and web.

## Features

- Create an instance and invite members (local prototype logic).
- Manage menus and their participants.
- Sections for petit-déjeuner, apéro and communs with items.
- Each item has name, details, quantity, unit and rayon.
- Preconfigured items list.

## Development

```bash
npm install
npm start
```

Run on the web with:

```bash
npm run web
```

There are currently no automated tests.
